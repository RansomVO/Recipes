InstaPot
While on his mission, Josh learned about a cooking appliance named "Instant Pot", but we misread the name. Even after we discovered our mistake, we kept using the same name. (Thus the name of this section) But we LOVE the things we can do with our "InstaPot"!
It can be used to sauté, brown, cook rice, slow cook, pressure cook, etc. As a result, it is super handy if you want to cook a dish that may need multiple modes for different stages.
